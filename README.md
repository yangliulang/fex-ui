# fex-ui

## 项目安装依赖

```
npm install
```

### 从 develop 模式启动项目，热更新

```
npm run serve
```

### 打包线上项目

```
npm run build
```

### 构建 fex-ui 库 -> npm publish from ./lib 文件夹

```
npm run lib
```

### Lints and fixes files

```
npm run lint
```
